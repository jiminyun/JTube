# JTube

Cloning YouTube with Vanilla javascript and Nodejs

## Built With

- Frontend
  - Vanila JavaScript
  - SaSS
  - ES6
  - Webpack
  - Pug
- Backend
  - MongoDB
  - Nodejs
  - Express
  - AWS S3
  - Heroku

## Getting Started

```
git clone https://github.com/jiminyun/JTube
npm install

npm dev:server
npm dev:assets
```
